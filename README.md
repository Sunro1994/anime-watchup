# anime-watchup

친구 그룹용 애니메이션 시청 기록 PWA.

## Stack
Next.js 15 · TypeScript · Tailwind · Supabase · AniList GraphQL

## Local Dev
```bash
pnpm install
cp .env.example .env.local      # 값 채우기
pnpm dev
```
